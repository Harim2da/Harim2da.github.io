1) 피보나치 수열을 2가지 버전으로 구현하는 방법을 배웠다.
	* 피보나치 수열이란 ? 
		* A의 n승 = A의 n-1 승 + A의 n-2승
		* 즉, n 번째 값은 직전의 값과 전전의 값의 합

	```java

// for문 사용하면서 구하는 방법
int size = 100
int[] arr = new int[size];
arr[1] = 1;  // n = 3 일 때부터 적용 가능함
arr[2] = 1;

for(int i = 3; i <= size; i++) {
	arr[i] = arr[i-1] + arr[i-2];
}

for(int i : arr) {
	System.out.print(i);
}


// 두 번째 방법
int n1 = 1;
int n2 = 1;

System.out.print(n1 + " ");
System.out.print(n2 + " ");

for(int i = 3; i <= size; i++) {
	int currNum = n1 + n2;
	System.out.print(currNum + " ");

	int n1 = n2;
	int n2 = currNum;
}


```


2) 최빈수 구하기
	- 주어진 숫자들 중 가장 많이 출현한 수를 구하기

	```java
	Scanner scan = new Scanner(System.in);

	int[] inputNum = new int[10];
	for(int i = 0; i < 10; i++) {
		inputNum[i] = scan.nextIn();
	}

	// input 예시 - 1 2 2 3 1 4 2 2 4 3
	// 최빈수는 2 (4번 등장)
	int[] mode = new int[10];

	/*
	* index : 출한현 숫자
	* index 값 : index가 몇 번 나왔는지 저장
	* ex) mode[3] = 5  -> '3'이 5번 나왔음 
	*/ 
	for(int i = 0; i < 10; i++) {
		mode[inputNum[i]]++;
		// mode의 0번째에 있는 숫자는 제일 처음 입력받은 숫자인데, 그 값에서 하나씩 카운트를 올려줌
	}
	int modeNum = 0; // 최빈수
	int modeCnt = 0; // 최빈수의 출현 횟수
	for(int i = 0; i < 10; i++) {
		if(modeCnt < mode[i]) {
			modeCnt = mode[i];
			modeNum = 1;
		}
	}
	System.out.println("최빈수는 : " + modeNum + " cnt는 : " + modeCnt);

	
```
